# OverSite Frontend

The OverSite frontend is a React-based IDE designed for technical assessments. It provides a VS Code-like environment where candidates can write code, run tests, and interact with an AI assistant.

## Key Features
- **Monaco Editor:** Industry-standard code editing experience.
- **Embedded AI Chat:** Integrated assistant for real-time help.
- **Telemetry Stream:** Debounced logging of editor events, execution results, and AI interactions.
- **Phased Workflow:** Guides candidates through Orientation, Implementation, and Verification phases.

## Technology Stack
- **React 19** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** for styling
- **Lucide React** for iconography
- **Vitest** for unit and component testing
